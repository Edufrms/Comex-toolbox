DataComex Tutorial

Detailed consultation of Spanish exports and imports by product, country, region, and company

1. What is DataComex?

DataComex is a tool developed by the Spanish Secretary of State for Trade that provides official statistics on Spanish foreign trade with a high level of detail. It allows you to:

  • Access monthly, annual, and accumulated data
  • Filter by product (HS), country, autonomous community, or province
  • See the number of exporting or importing companies
  • Download the data in Excel
Note: It is free, requires no registration, and is updated monthly with data from the Tax Agency.

2. Access to the system

  1. Access: datacomex.comercio.es
  2. Click on “Estadísticas de comercio exterior” (Foreign trade statistics)
  3. Select the type of query you want to perform

3. Types of queries available

  • Consulta por país: ¿qué productos exporta España a un mercado concreto? (Query by country: What products does Spain export to a specific market?)
  • Consulta por producto: ¿a qué países se exporta un código arancelario determinado? (Query by product: To which countries is a specific tariff code exported?)
  • Consulta por comunidad: ¿qué exporta Castilla y León, Andalucía, etc.? (Query by community: What does Castilla y León, Andalucía, etc., export?)
  • Consulta por empresa: requiere acceso especial (Query by company: Requires special access)

Each query can be broken down by year, period, FOB/CIF value, and more.

4. How to perform a search

  1. Select the type of query (e.g., by product)
  2. Enter the HS code or select it from the menu
  3. Filter by year, community, country, or grouping
  4. Click on “Consultar” (Query)

You will see an interactive table with exports/imports by value, volume, number of transactions, etc.

5. Exporting results

  • Use the “Exportar a Excel” (Export to Excel) button to download the data
  • You can save favorite filters for periodic analysis
  • The results can also be displayed in graphic format

6. Practical example

Example: Consulting wine exports from Spain to the U.S. in 2023

  1. Tipo de consulta: “Por producto” (Type of query: "By product")
  2. Producto: “Vino” (Wine) (HS 2204)
  3. Año: 2023 (Year: 2023)
  4. País destino: Estados Unidos (Destination country: United States)
  5. Resultado: se muestra volumen exportado, valor FOB y número de operaciones (Result: Exported volume, FOB value, and number of transactions are shown)

7. Usage recommendations

  • Use DataComex to prepare SWOT analyses, country sheets, justifications for aid, or market studies
  • Complement with TradeMap and Export Potential Map for a comparative analysis
  • Remember to check the new data monthly (published between the 20th-25th)
