Tutorial de Recursos para Empresas - Junta de Castilla y León

Accede a ayudas, servicios, formación y apoyo para empresas desde la web oficial

1. ¿Qué es este portal?

La web oficial de la Junta de Castilla y León para empresas centraliza información sobre:

  • Subvenciones y ayudas regionales
  • Apoyo a la internacionalización y exportación
  • Digitalización, innovación y sostenibilidad
  • Formación y asesoramiento para emprendedores
Nota: Está dirigida a autónomos, pymes y grandes empresas con sede en Castilla y León.

2. Acceso al sitio web

  1. Visita: empresas.jcyl.es
  2. Accede al menú lateral izquierdo para explorar áreas clave
  3. También puedes utilizar el buscador general de la web

3. Secciones destacadas

  • Ayudas y subvenciones: convocatorias abiertas y anteriores
  • Internacionalización: apoyos para ferias, misiones y asesoramiento ICE
  • I+D+i: incentivos para digitalización, industria 4.0 y sostenibilidad
  • Formación: programas gratuitos y formación especializada
Tip: Usa los filtros por sector, provincia o tipo de ayuda para encontrar programas relevantes.

4. Cómo solicitar una ayuda

  1. Accede a la sección “Ayudas y subvenciones”
  2. Selecciona la convocatoria vigente de tu interés
  3. Descarga las bases y requisitos
  4. Accede al enlace “Tramitación electrónica”
  5. Necesitarás un certificado digital o Cl@ve

Algunas convocatorias son gestionadas directamente por el ICE.

5. Ejemplo práctico

Ejemplo: Solicitar ayuda para asistencia a una feria internacional

  1. Ir a “Internacionalización”
  2. Seleccionar “Convocatoria para ferias internacionales 2025”
  3. Revisar requisitos: ser pyme, tener producto exportable, etc.
  4. Descargar modelo de solicitud
  5. Presentar electrónicamente con firma digital

6. Recomendaciones

  • Suscríbete al boletín del ICE para no perder convocatorias
  • Consulta el apartado de formación para aprovechar talleres gratuitos
  • Utiliza este portal como base para complementar con ICEX, CEOE o Cámaras
