Tutorial de UN Comtrade Plus

Accede a estadísticas globales de comercio internacional por producto, país y año desde fuentes oficiales de la ONU

1. ¿Qué es UN Comtrade Plus?

Es la plataforma de datos comerciales de Naciones Unidas, que proporciona información oficial y armonizada sobre importaciones y exportaciones entre países, incluyendo:

  • Flujos de comercio por producto (códigos HS/SITC)
  • Datos por país emisor y receptor
  • Estadísticas anuales y mensuales
  • Visualizaciones y descarga de bases de datos
Nota: La herramienta permite acceder a millones de registros y generar consultas personalizadas con filtros avanzados.

2. ¿Cómo acceder?

  1. Visita: comtradeplus.un.org
  2. No es necesario registrarse para consultar datos públicos
  3. Usa el botón “Build Query” para crear una consulta personalizada

3. Cómo construir una consulta

  1. Haz clic en “Build Query”
  2. Selecciona el tipo de flujo: importaciones, exportaciones o ambos
  3. Elige los países (reporter y partner)
  4. Selecciona el sistema de clasificación (HS 2017, SITC, etc.)
  5. Introduce el producto (por nombre o código)
  6. Indica los años y haz clic en “Run Query”

El sistema te mostrará una tabla interactiva con los resultados.

4. Visualización y descarga

  • Los resultados se pueden ver como tabla, gráfico o mapa
  • Puedes descargar los datos en CSV o JSON
  • También puedes copiar enlaces para compartir tus consultas
Tip: Usa los filtros por valor, unidad o código de producto para obtener insights más precisos.

5. Ejemplo práctico

Ejemplo: Exportaciones de cacao de Costa de Marfil a España

  1. Tipo de flujo: Exportaciones
  2. País reportero: Costa de Marfil
  3. País socio: España
  4. Código producto HS: 1801 (cacao en grano)
  5. Año: 2022
  6. Haz clic en “Run Query”

El sistema mostrará valor FOB, volumen, país de destino y tendencias por año.

6. Recomendaciones finales

  • Comtrade es ideal para informes de análisis de competencia o flujos globales
  • Úsalo junto con TradeMap para enriquecer tus gráficos o justificar estrategias de mercado
  • Verifica siempre las unidades de medida y la cobertura de datos por país
